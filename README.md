# TLS-HCC-Code
Code for HCC TLS multi-omics analysis
This repository contains analysis scripts used in the manuscript
“Multi-omics Spatial Profiling Defines Tertiary Lymphoid Structure Subtypes and a Cholangiocyte–Neutrophil Axis driving TLS Dysfunction in Hepatocellular Carcinoma”.

Scripts are provided as-is and correspond to the analyses used in the study.
